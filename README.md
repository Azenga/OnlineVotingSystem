## Install

```sh

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate:fresh --seed

php artisan serve

```

### Dependencies

 - Composer
    * laravel/ui
 - NPM
    * Bootstrap5.0.0-alpha1 !!
    * Vue Js
